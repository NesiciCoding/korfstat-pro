# Match Setup

The Match Setup screen is where you configure everything about a match before it begins. You reach it by clicking **"New Match"** on the Home Dashboard.

---

## Step 1: Assign the Match to a Season (Optional)

At the top of the Match Setup screen, you can optionally **assign this match to a season**. If you have created seasons in the [Season Manager](Season-Manager), they will appear in a dropdown here.

Assigning a match to a season means it will be included in the season standings table automatically. If you are just playing a friendly or a test match, you can skip this step.

---

## Step 2: Select a Match Profile

KorfStat Pro supports different rule sets and match formats. A **Match Profile** sets the:
- Duration of each half (e.g. 25 minutes for standard indoor, 12.5 minutes for beach korfball)
- Shot clock duration (e.g. 25 seconds standard)

Choose the profile that matches your competition rules from the dropdown.

> **Tip:** You can set a default half duration in [Settings](Settings) so you do not have to change it every time.

---

## Step 3: Set Up Teams

### Team Name and Color

For each team (Home and Away), enter:
- **Team Name** — the name that will appear on the scoreboard and in statistics.
- **Team Color** — used in charts, the field display, and overlays.

### Adding Players

Click **"Add Player"** to add players to each roster. For every player, enter:
- **Player Number** — their jersey number.
- **Player Name** — their name, which will appear in substitution popups and stats.
- **Gender** — Male or Female (used for visual differentiation on the strategy board and statistics).

You can add as many players as you need. You can also import a full roster from the [Club Manager](Club-Manager) if you have set up your clubs in advance.

---

## Step 4: Set the Starting Lineup

Korfball starts with half the players in **Attack** and half in **Defense**. Use the lineup section to mark which players start in which zone.

- Drag players into the Attack or Defense column, or use the toggle buttons next to each player's name.
- At the start of a standard match, each team has **4 players in Attack and 4 in Defense**.

---

## Step 5: Start the Match

When everything is set up, click the **"Start Match"** button. You will be taken directly to the [Match Tracker](Match-Tracker) screen.

> **Your data is safe!** All match setup data is saved automatically. If you accidentally refresh the page or close the browser, the match state will be restored when you re-open the app.

---

## Loading a Previous Match Setup

If you have run this fixture before, some teams may already exist in your Club Manager. You can quickly pre-fill the team details instead of typing everything from scratch every time.
