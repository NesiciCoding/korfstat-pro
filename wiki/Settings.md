# Settings

The Settings panel lets you customise how KorfStat Pro looks, sounds, and behaves. Open it by clicking the **gear icon** in the bottom-left corner of any screen.

---

## General

### Theme
Choose how the app looks:
- **Light** — a bright, white interface.
- **Dark** — a dark interface (great for low-light environments like indoor sports halls).
- **System** — automatically follows your device's light/dark mode setting.

### Language
Switch the interface language between:
- **English**
- **Nederlands (Dutch)**

### Show Timer in Browser Tab
When enabled, the game clock will appear in the browser tab title (e.g. `12:34 – KorfStat Pro`). Handy if you have multiple tabs open and want to glance at the clock without switching windows.

---

## Match Functions

### Game Effects (Sound)
Toggle the sound effects on or off. Sound effects play for events like goals and clock expiry.

### Match Type
Set the competition format:
- **Indoor (Standard)** — standard indoor Korfball rules.
- **Beach Korfball** — adjusted rules and timing for beach Korfball.

---

## Match Defaults

These settings are used as the default values when you set up a new match, saving you from changing them every time.

### Wear OS Control Mode
Sets the default mode for the referee's Wear OS smartwatch:
- **Read Only (Jury controls time)** — the watch displays data but the referee cannot change the clocks from their wrist.
- **Write (Referee controls time)** — the referee can start/stop the game clock and reset the shot clock from the watch.

### Half Duration
Set the default length of each half (in minutes):
- 10, 12.5, 15, 20, 25, or 30 minutes.

### Default Team Names
Pre-fill the Home and Away team names that appear in the Match Setup screen by default.

---

## Integrations

### Google Gemini API Key
Enter your API key from [Google AI Studio](https://makersuite.google.com/app/apikey) to unlock AI-powered features (Match Analysis, live commentary). Leave blank to skip.

### AI Model
Choose which Gemini model powers the AI features. See [Match Analysis](Match-Analysis) for a comparison of available models.

### Bitfocus Companion / Button-Box
Click **"Configure →"** to open the [Companion Integration](Companion-Integration) setup panel. This lets you connect KorfStat Pro to hardware button boxes and automation tools.

---

## Danger Zone

### Clear All Data
This button **permanently deletes all saved data** — all matches, clubs, seasons, saved plays, and settings. This cannot be undone.

> ⚠️ **Warning:** Only use this if you want to completely reset the app to factory defaults. Export any important data first!
