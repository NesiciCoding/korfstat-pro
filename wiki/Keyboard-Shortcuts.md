# Keyboard Shortcuts

KorfStat Pro supports a range of keyboard shortcuts to speed up your workflow during a match. These are particularly useful for experienced operators who want to keep their hands off the mouse as much as possible.

---

## General Shortcuts

| Shortcut | Action |
|---|---|
| `?` | Open the Shortcuts reference panel |
| `Esc` | Close any open modal or dialog |

---

## Match Tracker Shortcuts

| Shortcut | Action |
|---|---|
| `Space` | Toggle the Game Clock (start/stop) |
| `R` | Reset the Shot Clock to 25 seconds |
| `S` | Toggle the Shot Clock (start/stop) |
| `Z` or `Ctrl+Z` | Undo the last recorded event |
| `G` | Quick-record a goal for the home team |
| `H` | Quick-record a goal for the away team |

> **Note:** Shortcuts only work when you are not typing in a text box. If they are not responding, click somewhere on the field or scoreboard first to bring focus back to the app.

---

## Jury View Shortcuts

| Shortcut | Action |
|---|---|
| `Space` | Toggle the Game Clock |
| `R` | Reset the Shot Clock |
| `S` | Toggle the Shot Clock |

---

## Tips

- During a fast match, keeping one hand near the spacebar for the game clock is the most useful habit.
- Use `Z` to quickly undo if you accidentally record the wrong event — much faster than using the mouse.
- Assign team-specific actions to a hardware button box via [Bitfocus Companion](Companion-Integration) for even faster control.
