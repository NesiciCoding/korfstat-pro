# Live Stats View (Big Screen)

The Live Stats View is designed to be shown on a **large TV, monitor, or projector** at the venue. Spectators looking at this screen will see the live scoreboard and game clock at all times.

---

## Opening This View

From the Home Dashboard, click **"Live Stats (Big Screen)"**. Or open it directly with:
```
http://[your-ip]:3000/?view=LIVE
```

---

## What Spectators See

### Live Scoreboard
The team names and scores are displayed in large, clear text in the centre of the screen — always up to date.

### Game Clock
The current game time is displayed prominently.

### Shot Clock
The 25-second shot clock is also visible so spectators can follow the action.

### Animated Event Alerts
When a significant event happens (such as a **Goal** or a **Card**), a full-screen animation briefly appears on the big screen to highlight the moment. These animations are automatic — no extra action needed.

---

## How to Set Up the Big Screen

1. Connect a monitor, TV, or projector to the computer running KorfStat Pro.
2. Open a browser window on that second display.
3. Navigate to `http://localhost:3000/?view=LIVE` (or the network IP if using a different device).
4. Press **F11** to go fullscreen.

> **Tip:** You can also use a device like a Chromecast, Apple TV, or Smart TV's browser to access the view wirelessly.

---

## Keeping It in Sync

The Live Stats View automatically stays in sync with the Match Tracker and Jury View. Any goal, clock change, or event recorded anywhere in the system is reflected on the big screen instantly (within a fraction of a second).
